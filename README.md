# 📌 Project Overview

DP AI Assistant is a **voice-controlled desktop assistant built using Python**.  
It listens to user commands and performs tasks such as opening websites and responding with voice output.

The project demonstrates the integration of:

- Speech Recognition
- Text-to-Speech
- Python automation
- Web-based UI using Eel

---

# 🎯 Objective

The goal of this project is to build a **basic virtual assistant system** capable of understanding voice commands and executing tasks automatically.

This project helps demonstrate:

- Voice interaction with software
- Automation with Python
- Building simple AI assistant workflows

---

# ⚙️ Features

- 🎤 Voice command recognition
- 🔊 Text-to-speech responses
- 🌐 Open websites (Google, YouTube)
- 🖥 Simple desktop interface
- 🧠 Modular assistant architecture

---

# 🧠 How It Works

1. The assistant listens for a voice command.
2. SpeechRecognition converts audio into text.
3. The command is processed in Python.
4. The assistant performs the requested action.
5. Pyttsx3 generates a voice response.
